# Gest-o-de-Oficina
Sistema CLI em Java SE para gestão de oficinas mecânicas. Focado em Programação Orientada a Objetos (POO), aplica herança, polimorfismo e encapsulamento para gerenciar veículos, ordens de serviço e estoque. Utiliza Collections para manipulação de dados em memória e JDBC para persistência robusta em banco de dados PostgreSQL.
